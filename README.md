# Socionic
Приложение для прохождения теста по определению соционического типа

Стек:

+ Architecture MVVM
+ DataBinding
+ Navigation
+ DataBinding
+ MotionLayout
+ Animation
+ RecyclerView
+ Hilt (DI)
